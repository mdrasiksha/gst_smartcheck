WHAT IS THIS SOFTWARE?
------------------------------------------------------------
This software automatically extracts data from GST invoices and generates
a structured Excel report including:

• Invoice Number
• Invoice Date
• GST Number
• Taxable Amount
• CGST / SGST / IGST
• Final Amount
• Validation Status
• Confidence Score

Works with:
- Regular GST invoices
- Electrical / Industrial invoices
- SaaS invoices (MMT, ACT, KREDENT, etc.)
- Small value invoices

------------------------------------------------------------
SYSTEM REQUIREMENTS
------------------------------------------------------------
• Windows 10 or above (64-bit)
• Minimum 4GB RAM
• No internet required
• No Python installation required

------------------------------------------------------------
INSTALLATION STEPS (VERY IMPORTANT)
------------------------------------------------------------

1. Copy the file "InvoiceAutomationApp.zip" to:
   C:\tools\

2. Right click the ZIP file → Extract Here

3. You must see this folder:
   C:\tools\InvoiceAutomationApp

4. Inside the folder you should see:
   - app.py
   - main.py
   - run_app.bat
   - run_silent.vbs
   - other .py files

------------------------------------------------------------
HOW TO RUN THE SOFTWARE
------------------------------------------------------------

1. Double click:
   "Invoice Automation" shortcut (on Desktop)

OR

2. Go to:
   C:\tools\InvoiceAutomationApp
   Double click: run_silent.vbs

The application will open in your browser automatically.

------------------------------------------------------------
HOW TO USE
------------------------------------------------------------

1. Click "Browse" button
2. Select GST Invoice PDF file
3. Click "Upload"
4. Wait for processing
5. Download Excel file

------------------------------------------------------------
OUTPUT
------------------------------------------------------------

Processed Excel files will be saved in:
C:\tools\InvoiceAutomationApp\output\
